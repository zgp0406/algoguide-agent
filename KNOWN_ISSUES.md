# 当前问题记录

这里记录的是项目里已经确认过、但暂时还不放进 README 的问题和处理思路，方便后续继续修。

## 1. PDF 提取乱码

现象：

- 上传某些 PDF 时，提取出来的文本会出现乱码、符号错位或阅读顺序混乱。

常见原因：

- 扫描版 PDF，本质上是图片，直接抽文本效果很差
- 双栏排版、复杂公式、特殊字体编码会影响提取顺序
- 公式和正文混排时，纯文本抽取很容易失真

当前处理：

- 先做文本质量检测
- 尽量跳过公式块，只保留正文
- 质量太差时优先尝试 OCR 兜底
- 仍然不稳定时直接提示用户更换更清晰的文档

## 2. 旧 `sessions.json`

说明：

- 现在聊天记录的主存储是 `data/sessions_store.sqlite3`
- `data/sessions.json` 是早期版本留下的兼容文件，不是当前主存储

为什么还保留：

- 方便迁移旧会话
- 避免历史数据在升级过程中丢失

当前状态：

- 新的会话保存都写入 SQLite
- 旧 JSON 只作为迁移兼容使用
