# 前缀和笔记

前缀和适合快速回答“某一段区间的和是多少”。

## 一维前缀和

定义：

```text
pre[i] = nums[0] + nums[1] + ... + nums[i - 1]
```

那么区间 `[l, r]` 的和就是：

```text
sum(l, r) = pre[r + 1] - pre[l]
```

## 作用

- 快速求区间和
- 适合大量重复查询
- 是很多题的基础工具

## 常见场景

- 子数组和
- 区间统计
- 二维前缀和

## 二维前缀和

二维数组里也可以做前缀和，用来快速求子矩阵和。

如果用户问“某个矩阵区域的和”，可以先想到二维前缀和。
