# 最大子矩阵和

最大子矩阵和是二维数组中的经典题，目标是找出一个连续子矩阵，使它的元素和最大。

## 基本思路

常见做法是把二维问题降成一维问题：

1. 固定上边界和下边界
2. 把中间每一列累加成一个一维数组
3. 对这个一维数组求最大子数组和

这样就把问题转成了“二维枚举 + 一维最大子段和”。

## 一维部分

一维最大子数组和可以用 Kadane 算法：

```text
cur = max(x, cur + x)
best = max(best, cur)
```

## 复杂度

- 时间复杂度：`O(n^3)` 或 `O(n^2 * m)`，取决于矩阵大小和枚举方式
- 空间复杂度：`O(m)`

## 讲题建议

如果用户问“最大子矩阵和怎么做”，可以按这个结构回答：

1. 先说把二维问题变成一维
2. 再说为什么可以固定上下边界
3. 最后给出 Kadane 的一维处理

## 容易出错的地方

- 把“最大子矩阵和”误写成“最大子序列和”
- 忘了处理全负数情况
- 行列方向搞反
