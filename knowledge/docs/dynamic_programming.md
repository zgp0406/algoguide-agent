# 动态规划入门笔记

动态规划适合处理“重复子问题很多，而且可以从小问题推到大问题”的题目。

## 四步法

1. 定义状态
2. 写出转移方程
3. 确定初始值
4. 决定遍历顺序

## 例子：打家劫舍

定义 `dp[i]` 表示前 `i` 间房子能偷到的最大金额。

转移方程：

```text
dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
```

含义是：

- 不偷第 `i` 间，继承前一个状态
- 偷第 `i` 间，就不能偷 `i - 1`，要加上 `i - 2` 的结果

## 常见误区

- 状态定义不清楚
- 转移方程漏掉边界
- 先写代码，后想状态

## 回答模板

讲动态规划题时，建议按这个顺序讲：

1. 先说直觉
2. 再说状态定义
3. 再说转移方程
4. 最后说初始化和代码
