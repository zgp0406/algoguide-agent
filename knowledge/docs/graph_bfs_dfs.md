# 图论基础：BFS 和 DFS

图论题里，BFS 和 DFS 是最基础的两种遍历方式。

## BFS

广度优先搜索，适合：

- 求最短路径
- 分层遍历
- 找离起点最近的答案

核心特点：

- 用队列
- 一层一层扩展

## DFS

深度优先搜索，适合：

- 搜索所有可能
- 回溯
- 连通块
- 树结构遍历

核心特点：

- 先深入，再回退
- 常用递归或栈实现

## 讲解顺序

遇到图论题时，可以先问自己：

1. 是不是要找最短路径？如果是，优先想 BFS
2. 是不是要穷举路径或组合？如果是，优先想 DFS

## 常见误区

- BFS 和 DFS 都只会写模板，不会判断场景
- 没有区分“图”和“树”
- 忘了 visited，导致重复访问
